## Project Overview

NutriPlan Campus is a mobile application developed using Flutter and Firebase to help schools manage student nutrition information, dietary requirements, meal planning, and reports.

The system enables administrators to register students, assign diet categories, manage nutrition records, and access real-time information through a modern mobile interface.

## Installation

1. Clone the repository

```bash
git clone https://github.com/roxaebkb/nutriplan-africa.git
```

2. Navigate to the project directory

```bash
cd nutriplan-africa
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
